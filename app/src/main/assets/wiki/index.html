Installation (Steam)

Download game files

Zomdroid requires Project Zomboid files for Linux in order to run. If you're using Linux, simply download the game as usual through the Steam client. If you're using Windows or macOS, follow these steps::
  1. Open the Steam console. One way to do this is by typing steam://open/console into your browserâ€™s address bar
  2. Enter the following task: download_depot 108600 108603. This will download the latest stable version of Project Zomboid for Linux. If you want to download a specific version, you'll also need to provide a manifest id. For example, to download the unstable version from March 24, 2025, use: download_depot 108600 108603 2006259071304963476. You can find a list of all manifests on the SteamDB website (not affiliated with Steam or Valve)

Add files to a .zip archive

Navigate to folder containing game files (folder with ProjectZomboid64 executable). On Linux it will most likely be located at ~/.local/share/Steam/steamapps/common/ProjectZomboid/projectzomboid. On Windows or macOS you need to note the depot download directory from the previous step (it will appear in the console once the download is finished). E.g. on Windows it is usually C:\Program Files (x86)\Steam\steamapps\content\app_108600\depot_108603\projectzomboid. Select all contents and add them to a .zip archive

Copy the .zip archive to your device

Copy the .zip archive from the previous step to any folder on your android device. Further gameInstance will be handled by Zomdroid application.
